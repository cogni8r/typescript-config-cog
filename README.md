# TypeScrip Config

[TypeScript](https://www.typescriptlang.org) configuration file for personal projects.

## Usage

**Install**:

```shell
$ npm install --save-dev typescript typescript-config-cog
```

**Edit `tsconfig.json`**:

```jsonc
{
  // ...
  "extends": "typescript-config-cog/tsconfig.json",
}
```
